# i18n-lipsum

`i18n-lipsum` provides Lorem Ipsum-style test strings in a range of different languages for i18n testing as an NPM package.

## Supported languages

- English / Latin
- Czech
- Norwegian / Danish
- French
- Hungarian
- Icelandic
- Italian
- Old English
- Polish
- Portuguese
- Spanish
- Swedish
- Turkish
- Yugoslavian
- Lëtzebuergesch
- Russian
- Greek
- Armenian
- Georgian
- Hindi
- Chinese
- Japanese
- Jugemu
- Korean
- Arabic
- Hebrew
- L33tspeak
- Morse code
- Esperanto
- Interlingua
- Quenya
- Slovio
- Sona
- Tokipona
- Volapük

## Usage

```ts
import lipsums from "i18n-lipsum";
```

## Install

```sh
// If you use yarn
yarn add i18n-lipsum

// If you use NPM
npm install i18n-lipsum
```
